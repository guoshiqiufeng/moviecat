# 豆瓣电影
	基于Bootstrap、Angular.js进行开发
	使用豆瓣api获得数据

##开始使用
    -安装npm、git
    -git clone https://github.com/guoshiqiufeng/moviecat.git
    -使用终端进入项目所在目录
    -npm start
